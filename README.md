# 🤳 Insagram clone 📸

## ⚒️ Set Up

```
npm i
```

⚠️🙅‍♀️ Don't merege any branch without first this having been approved! 🙅‍♀️⚠️

⚠️ Use boxicons for any [icon](https://boxicons.com) as a **FONT** not web component  ⚠️

### 🏃‍♀️ Run project

```
npm run dev
```

It's works on [localhost:3500](http://localhost:3500)


### ⚗️ Run test

```
npm run test
```
