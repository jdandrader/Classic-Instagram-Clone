# 🤳 Insagram clone 📸

## ⚒️ Set Up

```
npm i
```

### 🏃‍♀️ Run project

```
npm run dev
```

It's works on [localhost:3500](http://localhost:3500)


### ⚗️ Run test

```
npm run test
```

- Jsonwebtoken
- Handlebars
- hbs
- Jest
- Mocha

- dotenv
- bcryptjs
